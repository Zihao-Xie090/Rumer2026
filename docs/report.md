# 《人工智能导论》大作业报告

题目：可解释的谣言检测

## 1. 任务简介

本项目基于谣言检测数据集，构建一个输入推文文本、输出谣言二分类结果和判断依据的模型。标签 `0` 表示非谣言，标签 `1` 表示谣言。

## 2. 数据集分析

训练集共有 2840 条样本，验证集共有 401 条样本。数据字段包括 `id`、`text`、`label` 和 `event`。本项目主要使用 `text` 字段进行文本分类，`label` 字段作为监督信号。

## 3. 方法设计

本项目采用 `TF-IDF + Logistic Regression` 作为 baseline。首先对文本进行预处理，包括 URL 替换、用户提及替换、小写化和空格归一化；然后提取 unigram 和 bigram 的 TF-IDF 特征；最后使用逻辑回归进行二分类。

## 4. 可解释性设计

逻辑回归模型可以为每个 TF-IDF 特征学习一个权重。对于单条输入文本，本项目计算文本中特征值与模型权重的乘积，选取对当前预测贡献最大的词语或短语，并据此生成自然语言解释。

## 5. 实验结果

训练后运行 `python -m src.evaluate`，将验证集上的准确率、精确率、召回率、F1 值和混淆矩阵写入 `outputs/metrics.json`。最终报告中应填入实际运行结果。

## 6. 案例分析

可从 `outputs/val_predictions.csv` 中选择若干样本，展示真实标签、预测标签、关键词和解释文本，分析模型判断是否合理。

## 7. 局限性与改进方向

当前方法主要依赖词语统计特征，难以充分理解复杂语义、讽刺表达和跨事件背景知识。后续可以尝试使用深度学习文本模型，或使用学校提供的大语言模型接口对判断依据进行进一步润色。
